Event recording and playback for late subscribers

1. Event History Recording

Enable history tracking for specific events

// Enable history tracking for events matching pattern
evem.enableHistory('user.*', { maxSize: 10 });

// Publish events - they will be recorded
evem.publish('user.action', {
    action: 'login',
    userId: 123
});

// View history
const history = evem.getHistory('user.action');
console.log(history); // Array of event records

2. Replay Last Event

Late subscribers automatically receive the most recent event

// Enable history first
evem.enableHistory('status.*', { maxSize: 5 });

// Publish an event
evem.publish('status.update', {
    status: 'online',
    server: 'main-01'
});

// Late subscriber gets the last event immediately
evem.subscribe('status.update', (data) => {
    console.log('Received:', data);
}, {
    replayLast: true  // Automatically receive most recent event
});

3. Replay Full History

New subscribers receive all historical events

// Enable history
evem.enableHistory('chat.*', { maxSize: 20 });

// Publish multiple events
evem.publish('chat.message', { user: 'Alice', text: 'Hello!' });
evem.publish('chat.message', { user: 'Bob', text: 'Hi!' });
evem.publish('chat.message', { user: 'Charlie', text: 'Hey!' });

// Late subscriber receives ALL historical events
evem.subscribe('chat.message', (data) => {
    console.log(`${data.user}: ${data.text}`);
}, {
    replayHistory: true  // Receive entire event history
});

4. Event Timeline

Visual representation of event history

// Event history visualization
const history = evem.getHistory('user.action');

history.forEach(record => {
    console.log(`[${new Date(record.timestamp).toLocaleTimeString()}]`,
                record.event,
                record.data);
});

// Example output:
// [10:30:45] user.action { action: 'login', userId: 123 }
// [10:31:02] user.action { action: 'update', userId: 123 }
// [10:31:15] user.action { action: 'logout', userId: 123 }

5. Late Subscriber Sync

Simulate a component joining late and catching up

// Complete late subscriber example
class ChatComponent {
    constructor() {
        // Component joins late but needs to catch up
        evem.subscribe('chat.message', (data) => {
            this.displayMessage(data);
        }, {
            replayHistory: true  // Get all missed messages
        });
    }

    displayMessage(data) {
        console.log(`${data.user}: ${data.text}`);
    }
}

// Publish some messages before component exists
evem.enableHistory('chat.*', { maxSize: 20 });
evem.publish('chat.message', { user: 'Alice', text: 'Hello!' });
evem.publish('chat.message', { user: 'Bob', text: 'Hi there!' });

// Component joins late and receives all history
const chat = new ChatComponent();
// Output: Alice: Hello!
//         Bob: Hi there!
